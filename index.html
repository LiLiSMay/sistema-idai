Sistema de Información para
Discapacidad Intelectual
y Cognitiva — IDAI

El Instituto de Adaptación Infantil (IDAI) de La Paz gestiona manualmente más de 17,000 expedientes físicos. Este proyecto propone el diseño de un sistema web y base de datos relacional para digitalizar y centralizar la información clínica, terapéutica y administrativa.
